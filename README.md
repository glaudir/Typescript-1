# Typescript 1.0
## Projeto para estudos da linguagem Typescript

# Rodando o projeto para teste
# Clone esse repositéorio
$ git clone <>

# Acesse a pasta do projeto no terminal/cmd
$ cd typescript-1

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm run start

# Entre com os dados no formulário
$ http://localhost:3000
